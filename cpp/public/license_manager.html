void set_machine_code(const std::string &code)¶

set the oneML machine code

This function can be called optionally to set a custom code. The code can be any alphanumerical sequency not longer than 100 characters and must not be empty. The code must be unique to guarantee correct licensing management. If this function is not called, oneML will automatically generate a unique code to assign to the machine.

Parameters

  • code: string of machine code
