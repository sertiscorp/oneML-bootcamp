Java Face Pad Example
^^^^^^^^^^^^^^^^^^^^^

.. only:: face_pad_rgb

    Java PadRGB Example
    """""""""""""""""""

    Initialize pad RGB model

    .. code-block:: java

        import org.sertiscorp.oneml.face.*;
        FacePad padRgb = new FacePad(PadType.Rgb);

    Initialize face detector model with licensing

    .. code-block:: java

        import org.sertiscorp.oneml.face.*;
        LicenseManager license_manager = new LicenseManager();
        license_manager.set_key("LICENSE_KEY_VALUE_HERE");
        license_manager.activate_key();
        FacePad padRgb = new FacePad(PadType.Rgb, license_manager);

    Initialize oneML's Utils

    .. code-block:: java
    
        Utils utils = new Utils();
    
    Initialize oneML's Utils with licensing
    
    .. code-block:: java
    
        Utils utils = new Utils(license_manager);
        
    Read input as BufferedImage

    .. code-block:: java

        BufferedImage img1 = ImageIO.read(image_path);
        ImageBatch batch1 = new ImageBatch().add(img1).add(img1);

    Read input as oneML's Image

    .. code-block:: java
            
        Image img1 = utils.readImageCV(file_path1);

    Run the model

    .. code-block:: java

        FacePadResultList rgbResults = padRgb.classify(batch1);

    or

    .. code-block:: java

        FacePadResult rgbResult = padRgb.classify(img1);

    And get the output

    .. code-block:: java

        float spoof_prob = output.get_spoof_prob();
        boolean is_spoof = output.is_spoof();
        int status = output.get_return_status();
