.. _cpp_face_landmark_example:

C++ Face Landmark Example
^^^^^^^^^^^^^^^^^^^^^^^^^

Initialize face landmark model

.. code-block:: c++

    oneML::face::FaceLandmarkDetector landmark_detector;

Initialize face landmark model with licensing

.. code-block:: c++

    oneML::LicenseManager license_manager;
    license_manager.set_key("LICENSE_KEY_VALUE_HERE");
    license_manager.activate_key();
    oneML::face::FaceLandmarkDetector landmark_detector(license_manager);

Initialize oneML's Utils

.. code-block:: c++

    oneML::Utils utils;

Initialize oneML's Utils with licensing

.. code-block:: c++

    oneML::Utils utils(license_manager);

Create oneML's MultiImage input and outputs object

.. code-block:: c++

    oneML::MultiImage inputs{img1, img2, ...  img4};
    std::vector<oneML::face::FaceLandmarkResult> outputs;

Create oneML's Image input and output object

.. code-block:: c++

    oneML::Image img;
    utils.read_image_cv(path, img);
    oneML::face::FaceLandmarkResult output;

Run the model

.. code-block:: c++

    landmark_detector.detect(inputs, outputs);

or

.. code-block:: c++

    landmark_detector.detect(img, output);
