Python Vehicle Detector Example
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Initialize vehicle detector model

.. code-block:: python

    from oneML import alprAPI as api
    detector = api.VehicleDetector()

Initialize vehicle detector model with licensing

.. code-block:: python

    from oneML import alprAPI as api
    license_manager = api.LicenseManager()
    license_manager.set_key("LICENSE_KEY_VALUE_HERE")
    license_manager.activate_key()
    detector = api.VehicleDetector(license_manager)

Read input as numpy array

.. code-block:: python

    img = np.array(cv2.cvtColor(cv2.imread(image_path), cv2.COLOR_BGR2RGB), dtype=np.uint8)
    imgs = np.expand_dims(img, axis=0)

Run the model

.. code-block:: python

    results = detector.detect(imgs)

or

.. code-block:: python

    result = detector.detect(img)
