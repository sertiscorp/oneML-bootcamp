Welcome to oneML's documentation!
=======================================

oneML is a fully-flegdged C++ SDK proving APIs for a number of different
AI/ML applications that can be deployed on any target (CPU, GPU, DSP) and
platform (android, iOS, embedded linux, linux, windows).
